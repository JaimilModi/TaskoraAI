## ✨ Overview

**Taskora AI** is an AI-powered productivity and content creation platform that provides powerful tools to generate content, enhance images, and review documents using artificial intelligence.

It combines multiple AI utilities into one clean, modern web application to boost creativity and efficiency.

---

## 🧠 Core Features

### 📝 AI Article Writer
Generate high-quality, engaging, and well-structured articles on any topic using advanced AI writing technology.

### 📰 Blog Title Generator
Create catchy, SEO-friendly blog titles instantly with AI-powered suggestions.

### 🎨 AI Image Generation
Generate stunning AI-powered visuals from text prompts and bring your ideas to life.

### 🖼 Background Removal
Effortlessly remove image backgrounds using AI-driven processing.

### ✂️ Object Removal
Seamlessly remove unwanted objects from images with intelligent AI editing.

### 📄 PDF Reviewer
Upload your resume or any PDF document and receive AI-powered feedback to improve quality and impact.

## 🛠 Setup & Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/JaimilModi/TaskoraAI.git
cd TaskoraAI
```

### 2️⃣ Install Frontend Dependencies

```bash
npm install
```

### 3️⃣ Install Backend Dependencies

```bash
cd client
npm install
```

### 4️⃣ Run the Project Locally

Start Frontend:

```bash
npm run dev
```

Start Backend:

```bash
cd client
node server.js
```
